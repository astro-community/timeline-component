This is the default experience.

The timeline can be updated programatically. For example, to change the start time of an event: 

timeline.tracks[1].events[0].startTime = '2022-01-01T00:30:00Z'

Or, to adjust the current level of zoom: 

timeline.zoom = 1.5

Or, to move the playhead forward by 1 hour: 

timeline.currentTime += 1000 * 60 * 60

Or, to move the start of the timeline back 1 hour: 

timeline.startTime -= 1000 * 60 * 60
